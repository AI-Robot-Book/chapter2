# turtlesim_launch
## 概要
２章のturtlesim用ローンチファイル

## インストール
- 端末で次のコマンドを実行してください．  
```
cd ~/airobot_ws/src/chapter2/turtlesim_launch
sudo cp ./mysim.launch.py /opt/ros/foxy/share/turtlesim/launch
```

## 実行  
- 端末で次のコマンドを実行してください．  
```
ros2 launch turtlesim mysim.launch.py
```
